SETUP GUIDE

How to setup project

step 1: Unzip the workspace from the folder :/ibm/sample Example : G:/ibm/sample

step 2: Open the RSA (or) RAD change the workspace to project which is given in step 1

step 3: Run the ddl script to add database schemas and tables

step 4: Run Notifications.jar in the parent folder of the project using the commandline

            java -jar Notifications.jar

step 5:

         i)Run the file sample/WebContent/home.jsp on server

        ii)Choose the Test Environment from the list of servers.

        iii)Click ok.
